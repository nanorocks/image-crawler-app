About

As a student at a technical faculty of computer science, I was always curious about the web and my possibilities. I concluded that the web has no limit on how it will develop and how it will serve in the future. This project is a small part of the web that will serve as a high-resolution imaging service.
The background algorithm is inspired by the Google search engine itself today. I used Python and a number of modules to develop this project. There is not much to discover how this algorithm works, except I will say that the application code will be publicly available on GITHUB in order to increase the open-source world.
I encourage all students and people who have a passion for technology and web, to join in the development of advanced and modern applications that will offer us a simple solution and a better life for all of as. I wish you all the best.

Greetings from nanorocks

How it works?

ImagesApp is modern web scanner that find FREE HQ images around web. One more step to crawl the web. :)
